# 🎵 بوت الموسيقى العربي

أقوى بوت موسيقى بالعربي على تيليجرام - متوافق 100% مع Railway

---

## 🚀 خطوات النشر على Railway

### 1️⃣ إنشاء حساب MongoDB Atlas (مجاني)
1. اذهب إلى [mongodb.com/atlas](https://mongodb.com/atlas)
2. أنشئ حساباً مجانياً
3. أنشئ Cluster مجاني (M0)
4. من Security > Database Access أنشئ مستخدم
5. من Network Access أضف `0.0.0.0/0`
6. انسخ رابط الاتصال (Connection String)

### 2️⃣ الحصول على API Telegram
1. اذهب إلى [my.telegram.org](https://my.telegram.org)
2. سجل دخول برقم هاتفك
3. اضغط API Development Tools
4. انسخ `API ID` و `API Hash`

### 3️⃣ إنشاء البوت
1. افتح @BotFather على تيليجرام
2. أرسل `/newbot`
3. اختر اسماً ويوزرنيم
4. انسخ الـ Token

### 4️⃣ النشر على Railway
1. ارفع الملفات على GitHub
2. اذهب إلى [railway.app](https://railway.app)
3. New Project > Deploy from GitHub
4. اختر المستودع
5. اذهب إلى Variables وأضف:

```
API_ID          = رقمك من my.telegram.org
API_HASH        = هاشك من my.telegram.org  
BOT_TOKEN       = توكن البوت من BotFather
OWNER_ID        = معرفك الرقمي
BOT_USERNAME    = يوزرنيم البوت
MONGO_URI       = رابط MongoDB
SUPPORT_CHANNEL = رابط قناة الدعم
```

- البوت يحتاج **صلاحية الإدمين** في المجموعة
- يجب أن تكون هناك **مكالمة صوتية نشطة** في المجموعة

## 🐛 حل المشاكل الشائعة

**خطأ: pip command not found**
← تأكد من استخدام `python311Full` في nixpacks.toml ✅ (تم حله)

**خطأ: No active group call**
← ابدأ مكالمة صوتية في المجموعة أولاً

**خطأ: MongoDB connection**
← تأكد من صحة MONGO_URI وإضافة 0.0.0.0/0 في Network Access
